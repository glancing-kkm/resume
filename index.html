서비스 소개 및 사용 방법

이 도구는 취업 지원서 작성 시 자주 필요한 두 가지 작업인 증명사진 편집과 맞춤법 점검을 한 번에 제공합니다. 사진은 업로드 후 원하는 픽셀을 입력해 크롭 영역을 미리 확인할 수 있고, 맞춤법은 OpenAI API 기반 문맥 교정을 우선 사용합니다.

사진 편집 사용 순서

  1. 사진을 업로드하거나 드래그앤드롭합니다.
  2. 비율 프리셋 또는 원하는 가로/세로 픽셀을 입력합니다.
  3. 점선 프레임을 기준으로 사진을 드래그해 위치를 맞춥니다.
  4. 필요하면 최대 용량(KB)과 출력 형식을 지정한 뒤 변환합니다.

맞춤법 검사 사용 순서

  1. 원문을 입력합니다. 글자 수는 공백 포함 기준으로 바로 표시됩니다.
  2. 검사 버튼을 누르면 문장 교정안을 제시합니다.
  3. 표시된 교정안 중 원하는 항목만 클릭해 반영할 수 있습니다.

자주 묻는 질문

업로드한 사진이나 문장이 서버에 저장되나요?

기본적으로 업로드된 사진은 브라우저에서 처리되며 서버에 영구 저장하지 않습니다. 맞춤법 검사용 텍스트는 API 요청 처리 목적 범위에서만 사용됩니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.

교정 결과가 항상 100% 정확한가요?

문맥 기반 교정 특성상 오교정이 발생할 수 있습니다. 공식 제출 전에는 반드시 최종 검토를 권장합니다.

모바일에서도 사용할 수 있나요?

가능합니다. 모바일에서는 도구 영역이 세로 1열로 표시되어 화면 폭에 맞춰 동작합니다.

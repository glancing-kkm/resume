서비스 소개

Last updated: 2026-03-02

Resume Tools는 이력서 제출 과정에서 반복되는 편집 작업을 간단하게 처리하기 위해 운영하는 웹 도구입니다. 현재 제공 기능은 증명사진 크롭/용량 조절과 한국어 맞춤법 검사입니다.

운영 원칙

  • 도구 사용 목적과 기능을 명확하게 안내합니다.
  • 오해를 유발하는 과장 표현을 사용하지 않습니다.
  • 사용자가 업로드한 데이터는 처리 목적 범위 내에서만 사용합니다.
  • 서비스 정책 변경 시 문서의 최종 수정일을 갱신합니다.

면책 안내

자동 교정 결과는 참고용이며, 제출 전 최종 검토 책임은 사용자에게 있습니다. 서비스 중단, 외부 API 장애 등으로 일부 기능이 제한될 수 있습니다.
